BinSpaghetti

BinSpaghetti is an open-source software suite for different things involving numbers.

We have:

  • A binary/decimal/hexadecimal/octal convertor
  • Weight convertors
  • A program that will output all numbers between zero and a given number
  • A arithmetic calculator
  • An energy calculator
